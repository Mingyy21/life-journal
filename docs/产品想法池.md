# 产品想法池

> **状态**：远景规划，非MVP内容。每次迭代时翻出来对照，确保不偏离用户的原始意图。

---

## 用户原始需求汇总

1. 记录人生中遇到的问题及如何解决
2. AI帮助吸收记录感受并提供反馈
3. 人生课题模块：人际(友情/亲情/爱情)、人生(工作/学习)、自我(情绪/认知/健身)
4. 每次复盘有进度条，知晓对每个课题的看法
5. AI能感知情绪，针对板块分析（不是聊天机器人）
6. 手机和电脑双端使用
7. 云端加密存储
8. AI能感知情绪的，能理解认知的长期陪伴者

---

## AI记忆的特殊需求

9. **不是记录所有记录**：针对高频出现的记录提高优先级，低频记录反馈慢
10. **根据使用次数的向量决定调取记忆的速度**：不是全量，而是有优先级
11. **从多维度分析情感**：不是单一正/负标签
12. **多维度理解认知**：追踪认知深度的变化

## 两种记录模式（v0.2实现）

### 模式一：情绪消解式（AI对话引导）

**流程**：
```
情绪上来了 → 打开App → 跟AI语音/文字对话
  ↓
AI先接住情绪（共情、倾听、帮你命名感受）
  ↓
情绪平复后，AI引导进入理性视角
  ↓
对话结束 → AI自动提炼成结构化日记
```

**特点**：
- 入口是感性的、混沌的、情绪驱动的
- AI角色：先共情，后引导
- 产出：对话日志 + 结构化摘要

**待解决的设计问题**：
- 对话如何保留？→ 每次对话结束AI提炼成日记，对话记录可翻看
- 新对话和旧对话相关怎么处理？→ 系统提示"你之前聊过这个课题：XXX，要继续还是新开？"
- 对话列表如何组织？→ 按课题分类，左边显示历史对话标题

### 模式二：独立记录式（自由书写）

**流程**：
```
打开App → 自由写 → 可选标签 → 保存 → 
AI不主动分析 → 用户手动点"分析" → AI读取分析
```

**特点**：
- 入口是理性的、自主的
- AI角色：静默的图书馆员，需要时才干活
- 积攒的记录可以标记为"未复盘"

## 三个智能体架构（v0.3+实现）

### 1. 对话Agent
- **职责**：实时对话，情绪感知，共情回复
- **触发**：用户在对话模式下交互
- **能力**：
  - 听懂情绪状态(当前VAD值推断)
  - 先共情再引导理性
  - 对话结束自动提炼结构化记录
  - 与之前同课题的对话/日记做关联

### 2. 课题分析Agent
- **职责**：针对单一课题深度分析
- **触发**：用户主动请求 / 课题有新记录时
- **能力**：
  - 追踪该课题下所有记录的认知变化
  - 发现该课题下的思维模式和盲区
  - 生成课题进度报告
  - 对比不同时间段对同一课题的认知差异

### 3. 周期复盘Agent

| 子Agent | 触发周期 | 职责 |
|---------|---------|------|
| **日间复盘** | 每次写日记 | 情绪标注(VAD+Plutchik)、课题关联、快速反馈 |
| **周度洞察** | 每周日 | 本周情绪趋势、高频课题汇总、认知模式变化 |
| **月度深度** | 每月最后一天 | 课题进度更新、待解决问题识别、认知盲区提示 |
| **年度人格** | 每年12月 | 完整人格特质分析(大五人格推断)、年度成长报告、来年建议 |

### 4. 模式识别Agent（常驻）
- **职责**：跨课题跨时间扫描，发现用户注意不到的规律
- **触发**：后台持续运行（有新数据时触发）
- **能力**：
  - "你每次在'亲情'课题有进展后，'情绪'课题的焦虑维度都会下降"
  - "过去半年，你的认知模式从'自责型'逐渐转向'问题解决型'"
  - "你人生中三次重大转折都发生在秋天"

## 待解决问题清单

1. **手机端提醒模式**：课题久未复盘如何提醒？应该做成"置顶"还是"推送通知"还是"打开App时提示"？
2. **对话式记录的会话管理**：新对话和旧对话相关的处理逻辑
3. **云端同步具体方案**：国产云服务(AliOSS/COS) vs 自托管 vs WebDAV
4. **备份与恢复**：数据导出格式、导入兼容性
5. **多用户**：是否需要？如果需要，要不要做家庭版/伴侣版（共享某些课题）？
6. **日记间关联**：用户手动关联两篇日记（"这篇和那篇有关"），还是AI自动关联？
7. **写作模板**：除了自由书写，要不要提供引导模板（如ORID、KPT）？
8. **AI分析深度**：用户要不要追问AI？（分析卡片下方有追问输入框）

## 不做的事情（明确排除）

- 社交功能（分享、评论、关注）
- AI替代心理咨询师（只做陪伴+引导，不做治疗）
- 自动发朋友圈/社交媒体
- 情感评分排行榜
- 与其他健康App同步（运动、睡眠等）→ 后期可考虑Apple HealthKit集成
