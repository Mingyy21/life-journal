## 三、认知科学参考文献

12. **Conway (2005)** — Self-Memory System
    自传体记忆层级模型：生命故事→生命时期→一般事件→情节细节

13. **McGaugh (2000)** — Memory Modulation Hypothesis
    杏仁核-海马体交互，情绪唤醒度决定记忆编码优先级

14. **Tulving (1983)** — Encoding Specificity Principle
    记忆提取时的状态越接近编码时，越容易被调取

15. **Habermas & Bluck (2000)** — Autobiographical Reasoning
    纵向推理(生命叙事维度) + 横向推理(跨域模式维度)

16. **Singer & Bluck (2011)** — Self-Defining Memory
    自我定义记忆：高度情绪化、反复回忆、与核心人生课题相关

17. **Anderson (2004)** — ACT-R Cognitive Architecture
    陈述性记忆激活值：基础激活+扩散激活+部分匹配+噪声

18. **Bower (1981)** — Mood-Congruent Memory Effect
    情绪一致性记忆效应

19. **Nairne (2007)** — Survival Processing Effect
    记忆对"与生存相关的信息"有天然高优先级编码

20. **Schacter (2012)** — Episodic Future Thinking
    回忆过去和想象未来的脑区高度重叠

21. **McAdams (2001)** — Narrative Identity
    人通过"讲述自己的故事"来构建身份认同

22. **Richter-Levin & Akirav (2003)** — Tag & Capture
    情绪事件在睡眠中的选择性重放机制

23. **Barsalou (1999)** — Perceptual Symbol Systems
    长时记忆中的概念以框架(Frame)组织

## 十一、AI分析系统设计参考（2025前沿研究）

### A. VAD情感模型前沿

51. **PERT — Prompting-based Emotion Regression from Transformers** (2024)
    Bulla & Mongiovi, GoodIT '24
    用提示工程而非微调实现对VAD三维的回归预测，在EmoBank上达到SOTA

52. **VAD-Integrated BERT for Emotion Intensity** (SemEval-2025)
    Moreno Novoa et al.
    BERT嵌入+VAD特征融合，宏观F1从0.5715提升至0.7434
    **关键发现**：Dominance维度最难预测(Pearson r=0.672)，Fear/Surprise预测差

53. **EDAER — VAD知识增强的细粒度情绪识别** (2025)
    李等人，山西大学学报(CCIR 2024)
    在VAD空间用欧氏距离构建27类GoEmotions情绪分布，高斯核转换为分布分数
    **关键发现**：VAD空间关联能补偿稀有情绪类别的数据不足

54. **GPT-4o作为VAD情绪估计器** (ACII 2025)
    Narimatsu et al.
    GPT-4o对Valence估计接近人类，对Arousal中等，对Dominance表现差
    **启示**：LLM做VAD分析需针对Dominance维度专门优化Prompt

55. **LLM嵌入+集成回归——中文维度情感分析最优范式** (ROCLING 2025)
    TCU团队，Li & Lin
    DeepSeek嵌入配合SVR(SVR回归)，效价PCC=0.805，唤醒度MAE=0.759
    **当前最优方案**：LLM词嵌入 + 回归模型 + 多模型集成

### B. 多Agent架构设计

56. **Bunny — Multi-Agent Dialogue for Emotional Processing** (TU Delft, 2025)
    双Agent架构：Protective Bunny（情感验证）+ Objective Bunny（替代视角）
    6轮迭代原型 + 日记研究验证，77.5%参与度
    **设计张力**：探索性提问 vs 具体引导、对话节奏控制、用户对终点的掌控权

57. **POCKET-MIND — Dual-Prompt Framework for Journaling** (Sungkyunkwan, 2025)
    双提示框架：情绪探索Prompt + 目标达成Prompt
    一周30人研究，证明两个维度联动优于分离处理
    **核心洞察**：不应将情绪追踪与目标设定分离

58. **Self-Growing Memory Network for Elderly Self-Efficacy** (Guo et al., 2025)
    四组件架构：经验提取模型 + 记忆网络 + 聊天Agent + 自适应反馈环
    SUS=77.3, 自我效能得分=34.2
    CBT-informed回应 + 成功/失败经验双通道记忆

59. **Narrative Reconstruction & Semantic Analysis** (Wang & Wang, GCCE 2025)
    AI Agent检测重复语言和主题模式，在恰当时刻给予共情反馈
    **核心能力**：打破习惯性叙事循环，促进心理韧性

60. **Sophia — System 3 Meta-Layer for Persistent Agents** (2025)
    三层认知架构：System1(感知) → System2(推理) → System3(叙事身份+长期适应)
    重复操作推理步骤减少80%，高复杂度任务成功率提升40%
    **核心机制**：过程监督思维搜索 + 叙事记忆 + 用户/自我建模 + 混合奖励

### C. 多阶段分析Pipeline

61. **AutoLife — Multi-Layer Automatic Journaling** (ACM 2025)
    三层框架：分割(传感器数据→时间窗) → 融合(多上下文文本化) → 生成(LLM合成日记)
    **关键设计**：将复杂日志任务分解为可管理的子任务

62. **RAG + AI Journaling Pipeline** (2024-2025)
    标准流水线：原始条目→文档分割(500token块)→Embedding→pgvector(HNSW索引)→语义检索→LLM上下文感知分析
    Recall@3=0.77, Precision@3=0.80, MRR=1.0

63. **Agentic Multi-Step Analysis Loop** (2025)
    5工具Agent套件：journal_search / analyze_sentiment / trend_analysis / reflect / suggest_actions
    ReAct循环：Prompt → LLM → Tool calls → Execute → Observe → Repeat
    Llama 4 Scout 17B: 90% pass rate, 0.77 tool precision, 4.2 steps/query avg

### D. CBT + AI设计模式

64. **Cognitive Distortion Detection Architecture** (2025 MDPI系统综述, 25篇论文)
    15种认知扭曲分类，RoBERTa微调为主流方案
    **关键教训**：初版模型标记几乎所有条目都有扭曲→产生评判感
    **修正方案**：上下文感知提示逻辑，根据用户情绪状态调适应答

65. **隐私优先的数据架构** (2025共识)
    本地优先部署 + 设备端密钥加密 + 账号注销触发完整数据导出+永久删除
    Voice journaling关键原则：录音永不被存储，仅内存处理，立即清除

### E. 中文NLP心理学应用

66. **NTULAW — 中文医疗反思维度情感分析** (ROCLING 2025)
    多尺度BERT + 领域自适应（医生标注数据 + Chinese EmoBank混合）
    **核心挑战**：隐含情绪的效价系统性高估，唤醒度向中值回归

67. **BERT vs 词典——中文情绪文本分析** (中华心理学刊, 2025, Vol.67)
    尤譯霆 & 杨立行：BERT语义表征显著优于三种传统词典方法
    **原因**：BERT捕捉上下文语境，词典仅能统计孤立词频
    使用三层神经网络(非线性)替代线性回归建模语言-情绪映射

68. **中文心理健康LLM** (2025)
    hfl/chinese-roberta-wwm-ext在同济大学中文心理健康数据集上表现最佳
    指令微调LLMs + 类型验证组件的统一信息抽取框架

69. **机器学习模拟日记情绪判断** (张燕, 2025)
    152篇古代日记(黄庭坚《乙酉家乘》) + 三次主观标注(一致性87.5%)
    机器学习模型预测准确率87.9%，证明ML可复现人类对日记隐微情绪的阅读

### F. 反射式AI设计原则

70. **Generative Life Agent — 本地优先+可追溯人格演化** (2025)
    Ollama本地LLM + ChromaDB持久记忆 + Reflect-Evolve引擎
    **核心设计**：所有人格特质修改记录在可追溯JSON日志中
    用户数据主权 + 离线运行 + 持续身份感

71. **CBMAS — 认知行为建模与激活引导** (NeurIPS 2025)
    连续激活引导框架：追踪偏差在Transformer层的动态轨迹
    **提示设计启示**：提示本身就是引导干预，理解偏差在哪层激活可指导更鲁棒的提示工程
