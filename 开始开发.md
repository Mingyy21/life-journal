# 开始开发

## 给开发者的启动指南

### 1. 用 VSCode 打开此目录
```
D:\总文件夹\项目\人生手记
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置 API Key
将 `.env.example` 复制为 `.env.local`，填入你的 DeepSeek API Key：
```
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxx
DEEPSEEK_BASE_URL=https://api.deepseek.com
AI_MODEL=deepseek-chat
```
API Key 获取地址：https://platform.deepseek.com

### 4. 启动开发服务器
```bash
npm run dev
```

### 5. 浏览器打开
```
http://localhost:3000
```

## 首次启动后

### 数据库初始化
打开页面后，IndexedDB会自动创建 `LifeJournalDB` 数据库，并初始化8个预设课题：
- 人际：友情、亲情、爱情
- 人生：工作、学习
- 自我：情绪、认知、健身

### 模拟分析
如果未配置DeepSeek API Key，AI分析会使用模拟模式——基于关键词匹配给出基础情绪判断。配置API Key后即可使用完整AI分析。

### PWA安装
- **Android Chrome**：打开网站 → 菜单 → "添加到主屏幕"
- **iOS Safari**：打开网站 → 分享按钮 → "添加到主屏幕"
- **桌面Chrome**：地址栏右侧会出现安装图标
