# 文件夹大纲

## 根目录
```
人生手记/
├── README.md                    # 项目说明
├── CLAUDE.md                    # Claude Code协作规则
├── 文件夹大纲.md                 # 本文件：项目文件索引
├── 开始开发.md                   # 给开发者的启动指南
├── package.json                 # 依赖配置
├── tsconfig.json                # TypeScript配置
├── next.config.js               # Next.js配置
├── tailwind.config.ts           # Tailwind CSS主题配置
├── postcss.config.js            # PostCSS配置
├── .env.example                 # 环境变量模板
├── .env.local                   # 本地环境变量(不提交Git)
│
├── docs/                        # 📚 设计文档
│   ├── 00-项目总览.md            # 第一性原理、MVP边界、版本规划
│   ├── 01-产品设计.md            # 用户场景、交互流程、参考产品分析
│   ├── 02-技术架构.md            # 技术栈选型、项目结构、API设计
│   ├── 03-数据模型.md            # Dexie.js四表Schema、预设课题
│   ├── 04-AI分析系统.md          # VAD+Plutchik模型、Prompt设计、多Agent预留
│   ├── 05-UI原型设计.md          # 页面原型、配色方案、字体、响应式断点
│   ├── 06-开发计划.md            # 6个Phase详细任务、预估周期
│   ├── 07-测试策略.md            # 单元/集成/E2E测试、CI/CD
│   ├── 记忆模型设计.md           # 三层五机制记忆模型(远景v0.3+)
│   ├── 产品想法池.md             # 用户原始想法、远景功能、待解决问题
│   ├── 完整聊天记录.md           # 本项目从零到一的所有聊天记录
│   └── 参考资料索引.md           # 所有网络搜索来源整理
│
└── src/                         # 💻 源码
    ├── app/                     # Next.js App Router页面
    │   ├── layout.tsx           #   根布局(Head metadata + PWA配置)
    │   ├── page.tsx             #   首页(写作入口 + 筛选 + 列表)
    │   ├── globals.css          #   全局样式(配色/字体/动画/滚动条)
    │   ├── api/
    │   │   └── analyze/
    │   │       └── route.ts     #   AI分析API端点(POST)
    │   └── diary/
    │       └── [id]/
    │           └── page.tsx     #   日记详情页(内容+分析卡片)
    │
    ├── components/              # React组件
    │   ├── Layout.tsx           #   全局布局(Header+底部导航+移动端菜单)
    │   ├── DiaryEditor.tsx      #   日记编辑器(标题/内容/课题/标签)
    │   ├── DiaryList.tsx        #   日记列表(按日期分组+标签显示)
    │   ├── DiaryDetail.tsx      #   日记详情卡片(元信息+内容)
    │   ├── TagFilter.tsx        #   标签筛选器(课题+标签联动)
    │   └── AnalysisCard.tsx     #   AI分析卡片(情绪条+VAD+洞察+反馈)
    │
    ├── hooks/                   # React Hooks
    │   ├── useDatabase.ts      #   数据库初始化Hook
    │   └── useDiary.ts         #   日记CRUD操作Hook
    │
    ├── lib/                     # 核心库
    │   ├── db.ts               #   Dexie.js数据库(Schema+CRUD+预设数据)
    │   ├── ai.ts               #   AI分析接口(DeepSeek API+模拟分析)
    │   └── crypto.ts           #   客户端加密工具(v0.2预留)
    │
    └── types/                   # TypeScript类型定义
        └── index.ts            #   日记/Tag/Topic/AnalysisResult/VAD/EmotionLabels等类型
```
