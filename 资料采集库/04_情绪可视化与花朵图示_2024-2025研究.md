# 情绪可视化与花朵图示 — 2024-2025 研究综述

> 采集时间：2026-07-21
> 关键词：emotion visualization, flower petal diagram, Plutchik wheel, embodied emotion

---

## 1. Plutchik情绪花瓣图

*Nature* (2024) 情绪分析研究使用Plutchik轮盘：
- **每个基本情绪为一片花瓣**
- **花瓣长度** = 该情绪在所有话语中占比
- 放射状花瓣布局直观对比不同数据集的情绪画像
- 作为"基本情绪"可视化，配合细粒度极坐标直方图

→ **启示**：这是我们"周度情绪花瓣图"的直接理论基础。8种Plutchik情绪 → 8片花瓣，花瓣长度代表该情绪在一周内的出现频率。

## 2. Emotion Bouquet：物理花朵情绪表达 (2024)

**Mensch und Computer 2024** (Bauhaus-Universität Weimar)：
- 四朵**气动（充气）机器人花**，每朵代表一种情绪：
  - **红** = 愤怒、**蓝** = 悲伤、**绿** = 平静、**黄** = 快乐
- **颜色** → 情绪类型，**亮度** → 强度
- **花瓣形状**遵循心理学研究：正面/低唤醒情绪 → 圆形大花瓣；负面/高唤醒 → 小而有棱角的花瓣
- **呼吸式运动模式**反映生理反应：愤怒 → 快速强力充气；悲伤 → 缓慢不完全充气；快乐 → 快速完全充气；平静 → 适中缓慢充气

→ **启示**：可以给花瓣添加**动态呼吸效果**——不是静态图片，而是根据情绪强度有节奏地缩放。愤怒花瓣短而急促，平静花瓣长而舒缓。这让情绪数据有了"生命感"。

## 3. The Feelings Flower (2025)

六个花瓣的核心情绪分类：
- **mad**(生气) → 愤怒、嫉妒、烦躁、挑剔
- **sad**(悲伤) → 孤独、失望、遗憾、无助
- **scared**(害怕) → 焦虑、紧张、不安、恐慌
- **joyful**(快乐) → 兴奋、满足、感恩、自豪
- **calm**(平静) → 放松、安宁、专注、满足
- **strong**(强大) → 自信、勇敢、坚定、有力量

每个花瓣向外扩展为更细粒度的感受词。

核心机制：**情绪标签化**(affect labeling)——命名情绪通过激活前额叶调节脑区来降低情绪强度。

→ **启示**：情绪花瓣图不只是一张图，而是一个**情绪标签化练习工具**。用户看到花瓣时可以学到："哦，原来我今天的不安属于'害怕'家族"。

## 4. 真实花朵对心理的影响 (2025)

*Frontiers in Psychology* (2025)，158人对照实验：
- 不同花瓣类型和颜色直接影响EEG alpha/beta波
- 观看不同类型的花朵显著减少紧张、愤怒、疲劳、抑郁和恐慌
- **对称性 + 重复花瓣排列 + 强烈色彩** = 独特的具身情感连接

*Frontiers in Human Neuroscience* (2025.1) 对比真实花/画的花/曼陀罗：
- 真实花朵的对称性和花瓣排列比绘画/曼陀罗产生更强的具身情感连接
- **熟悉感（对称）+ 新鲜感（变化）**的结合在保持警觉的同时带来平静

→ **启示**：设计花瓣图时，应该使用**真实花朵的视觉质感**（柔和的渐变、有机的形状、微妙的阴影），而不是几何化的矢量图。这才能在视觉层面触发心理效益。

---

## 5. 周度情绪花瓣图的设计原则（总结）

结合以上研究，一个好的情绪花瓣图应该：

| 原则 | 来源 |
|------|------|
| 8片花瓣对应8种基本情绪 | Plutchik (Nature 2024) |
| 花瓣长度反映情绪频率 | Plutchik轮盘量化方法 |
| 花瓣形状：正面情绪圆润，负面情绪有棱角 | Emotion Bouquet (2024) |
| 动态呼吸效果（强度映射到缩放节奏） | Emotion Bouquet气动原理 |
| 从核心情绪标签向外展开细粒度词汇 | Feelings Flower (2025) |
| 使用有机的花瓣视觉质感而非几何矢量 | 花朵心理研究 (2025) |
| 色彩映射需符合心理学约定 | 红=愤怒, 蓝=悲伤, 黄=快乐, 绿=平静 |
